### 查看服务日志

**WVP-Pro:**
```bash

# 或
tail -f logs/wvp-*.log
```

**流媒体服务:**
```bash

```

**Redis:**
```bash
docker logs -f alldata-redis
```

**MySQL:**
```bash
docker logs -f alldata-mysql
```

## 🔧 管理命令

### 停止服务

**停止 WVP-Pro:**
```bash
# 查找进程
lsof -ti:18080

# 停止进程
kill $(lsof -ti:18080)
```

**停止流媒体服务:**
```bash
docker stop zlmediakit
```

**停止所有 Docker 服务:**
```bash
docker stop zlmediakit alldata-redis alldata-mysql
```

### 重启服务

**重启 WVP-Pro:**
```bash
# 停止
kill $(lsof -ti:18080)

# 启动
cd /Users/andyapple/Downloads/wvp-GB28181-pro
nohup java -jar -Dspring.profiles.active=dev \
  -Dfile.encoding=UTF-8 \
  -Xms512m -Xmx2048m \
  target/wvp-pro-2.7.4-12111153.jar \
  > logs/wvp-startup.log 2>&1 &
```

**重启流媒体服务:**
```bash
docker restart zlmediakit
```
