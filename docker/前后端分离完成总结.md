### 配置详情

#### 1. 前端构建配置 (`web/vue.config.js`)

```javascript
outputDir: "dist",  // 构建输出到dist目录
assetsDir: "static",  // 静态资源目录
```

#### 2. Spring Boot配置 (`application-docker.yml`)

```yaml
web:
    static:
        path: ${WEB_STATIC_PATH:/opt/wvp/static}
```

#### 3. WebMvcConfig配置

```java
// 默认路径：/opt/wvp/static（容器内路径，对应宿主机：/home/wvp/static）
String defaultPath = "/opt/wvp/static";
```

### 前端更新流程

1. **本地构建**
   ```bash
   cd web/
   npm run build:prod
   ```

2. **打包上传**
   ```bash
   tar czf /tmp/wvp-frontend.tar.gz -C web/dist .
   scp /tmp/wvp-frontend.tar.gz root@172.31.127.47:/tmp/
   ```

3. **远程部署**
   ```bash
   ssh root@172.31.127.47
   cd /home/wvp
   rm -rf static/*
   tar xzf /tmp/wvp-frontend.tar.gz -C static
   chmod -R 755 static
   ```

4. **验证更新**
   - 前端文件更新后，Spring Boot会自动读取新文件（无需重启容器）
   - 刷新浏览器即可看到更新
