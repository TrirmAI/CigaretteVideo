# WVP-Pro 和流媒体服务启动说明

## 🚀 快速启动方式

### 方式一：使用 Docker Compose（推荐，但需要先解决网络问题）

```bash
cd docker
docker-compose up -d
```

**如果遇到镜像拉取失败，可以：**
1. 配置 Docker 镜像加速器
2. 或使用方式二单独启动流媒体服务

### 方式二：单独启动流媒体服务（Docker）

```bash
# 启动 ZLMediaKit 流媒体服务
docker run -d --name zlmediakit \
  -p 10935:10935 \
  -p 5540:5540 \
  -p 10000:10000 \
  zlmediakit/zlmediakit:master
```

### 方式三：启动 WVP-Pro 服务（需要先启动 Redis、MySQL、流媒体服务）

#### 1. 确保依赖服务已启动

**Redis:**
```bash
# 使用 Docker
docker run -d --name redis -p 6379:6379 redis:latest

# 或使用本地 Redis
redis-server
```

**MySQL:**
```bash
# 使用 Docker
docker run -d --name mysql \
  -e MYSQL_ROOT_PASSWORD=123456 \
  -e MYSQL_DATABASE=wvp \
  -p 3306:3306 \
  mysql:8

# 或使用本地 MySQL
mysql.server start
```

**流媒体服务（ZLMediaKit）:**
```bash
docker run -d --name zlmediakit \
  -p 10935:10935 \
  -p 5540:5540 \
  -p 10000:10000 \
  zlmediakit/zlmediakit:master
```

#### 2. 启动 WVP-Pro

**如果已编译 jar 文件：**
```bash
java -jar -Dspring.profiles.active=dev target/wvp-pro-*.jar
```

**如果需要先编译：**
```bash
mvn clean package
java -jar -Dspring.profiles.active=dev target/wvp-pro-*.jar
```

## 📋 服务端口说明

- **WVP-Pro**: 18080
- **Redis**: 6379
- **MySQL**: 3306
- **ZLMediaKit RTMP**: 10935
- **ZLMediaKit RTSP**: 5540
- **ZLMediaKit RTP**: 10000

## ✅ 验证服务状态

### 检查 WVP-Pro
```bash
curl http://localhost:18080/openapi/v1/health
```

### 检查流媒体服务
```bash
docker logs zlmediakit
```

### 检查 Redis
```bash
docker exec -it redis redis-cli ping
# 应该返回: PONG
```

### 检查 MySQL
```bash
docker exec -it mysql mysql -uroot -p123456 -e "SHOW DATABASES;"
```

## 🔧 配置文件位置

- WVP-Pro 配置: `src/main/resources/application-dev.yml`
- 流媒体配置: `zlmediakit-config.ini`

## 📝 注意事项

1. **数据库初始化**: 首次启动需要执行数据库初始化脚本
   - MySQL: `数据库/2.7.4/初始化-mysql-2.7.4.sql`
   - PostgreSQL: `数据库/2.7.4/初始化-postgresql-kingbase-2.7.4.sql`

2. **配置文件**: 确保 `application-dev.yml` 中的数据库和 Redis 配置正确

3. **端口冲突**: 确保所需端口未被占用

4. **网络问题**: 如果 Docker 镜像拉取失败，可以：
   - 配置 Docker 镜像加速器
   - 使用国内镜像源
   - 手动拉取镜像

## 🛑 停止服务

### Docker Compose
```bash
cd docker
docker-compose down
```

### 单独停止
```bash
docker stop zlmediakit redis mysql
docker rm zlmediakit redis mysql
```
